# Prerequisites

To run this project, please follow these steps:

1. Download [Docker Desktop](https://www.docker.com/products/docker-desktop/).
2. Install and login to Docker Desktop.
3. Open the Docker Desktop application.
4. Open a terminal or command prompt in the IDE, navigate to the root directory of this project, and run the following command:
```bash
docker compose up --build
```

5. Open the browser and navigate to the following URL:
```
http://localhost:5173
```
